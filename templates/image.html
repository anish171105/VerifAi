Instructions for Text Extraction & Credibility Check

What is Text Extraction?

Text extraction is the process of retrieving meaningful text from various sources, such as images, documents, or web pages.

Steps to Use the Website:

  • Upload Image: Click the "Upload" button to choose the image from which you want to extract text.
  • Get Extracted Text: The system will automatically extract the text from the uploaded image.
  • Analyze Text: Enter the extracted text into the provided input field to analyze its credibility.
  • View Results: Check the results section for a detailed breakdown, including the credibility response and probability score.

Use the tool to analyze additional inputs as needed.
